# App Store Screenshots

Add the following screenshot files here before submitting to app stores:

| File | Dimensions | Used For |
|------|-----------|----------|
| `screenshot-mobile-1.png` | 1290×2796 | Google Play, iOS App Store (iPhone 15 Pro Max) |
| `screenshot-mobile-2.png` | 1290×2796 | Google Play, iOS App Store |
| `screenshot-desktop-1.png` | 1920×1080 | Google Play (tablet), PWA install prompt |

## Tips
- Show the RO list, Summary tab, and OCR scan feature
- Use real data / realistic demo data — no empty states
- Google Play requires at least 2 phone screenshots
- iOS App Store requires screenshots for each supported device size (iPhone 6.7" required)

## Tools
- Take screenshots directly on device/simulator
- Use Figma or Sketch with device frames for polished mockups
